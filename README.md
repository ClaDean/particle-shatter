# 粒子操控师 - 破碎版 (Particle Manipulator - Shatter Edition)

这是一个基于 WebGL (Three.js) 和 AI 手势识别 (MediaPipe) 的互动网页项目。通过摄像头捕捉手势，你可以实时操控 3D 粒子组成的形状，体验“捏合旋转”和“张手炸裂”的视觉效果。

## 🎮 玩法说明

1.  **打开链接**：使用手机浏览器（推荐 Chrome 或 Safari）打开项目链接。
2.  **允许权限**：点击“允许”使用摄像头（仅用于本地手势识别，不会上传画面）。
3.  **放置手机**：将手机固定在桌面上，或者让朋友帮忙拿着。
4.  **展示手势**：站在距离手机约 0.5 米处，确保摄像头能拍到你的单手。
    *   **🖐 张开手掌**：粒子会瞬间崩解炸裂，向四周飞散。
    *   **👌 捏合手指 (食指+拇指)**：粒子会迅速重组，你可以移动手来旋转模型。
    *   **✨ 悬停**：手掌自然放松，粒子会悬浮在空中。
5.  **更多功能**：
    *   点击右侧菜单切换形状（爱心、土星、螺旋等）。
    *   开启“幻彩流光”体验彩虹特效。
    *   (高级) 点击“AI 造物”输入描述，让 AI 生成独一无二的粒子形状。

## 如何部署 

为了让朋友在手机上玩，你需要将项目部署到支持 HTTPS 的静态托管服务上。

1.  **创建仓库**：在 GitHub 上新建一个仓库（例如 `particle-shatter`）。
2.  **上传代码**：将本文件夹内的所有文件 (`index.html`, `README.md` 等) 上传到该仓库。
3.  **开启 Pages**：
    *   进入仓库页面 -> **Settings** (设置)。
    *   在左侧菜单找到 **Pages**。
    *   在 **Build and deployment** -> **Branch** 下，选择 `main` (或 `master`) 分支，点击 **Save**。
4.  **获取链接**：等待几分钟后，刷新页面，顶部会显示你的项目链接 (例如 `https://yourname.github.io/particle-shatter/`)。
5.  **分享**：把这个链接发给朋友即可！


## 📱 移动端优化说明

本项目已针对移动端进行了优化：
*   自动检测移动设备并降低粒子数量 (9000 -> 2000) 以保证流畅度。
*   降低了渲染分辨率以减少发热。
*   增加了 GPU 加速失败时的 CPU 回退机制，兼容更多机型。
